# Vue 2.7 + Vite 前端工程化项目

开箱即用的模板项目，集成主流技术栈

## 基础搭建

推荐使用 [pnpm](https://pnpm.io/zh/) 来作为本项目的包管理工具。

```bash
# 安装依赖
pnpm i

# 运行
pnpm run dev

# 打包
pnpm run build

# 打包文件预览
pnpm run preview
```


### 插件安装

此脚手架必须安以下依赖才能保证，代码自动格式化的有效运行：

- Vetur
- EditorConfig for VS Code
- ESLint
- Stylelint
- UnoCss
